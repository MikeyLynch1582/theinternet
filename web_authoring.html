Webpage Builders

Notepad

Notepad

Notepad is a free webpage builder which you use coding to create a website. This is the most basic version of a website builder. One of the main positives of notepad is that it is free and built into every Microsoft computer but it doesn’t show you where coding in wrong and doesn’t show you the difference in the styles and the content.

Notepad++

Notepad++

Notepad++ is a free webpage builder which you use to coding to create a website. This a better then notepad because this shows you where you have errors and is colour coordinated. Also allows you to but in comments so you know if you need to change anything. This is more advanced program then notepad but is less effective then Dreamweaver.

Dreamweaver

Dreamweaver

Dreamweaver is a website builder program which is owned by Adobe. This is highest quality of a product to produce and create websites. Dreamweaver has two different panels, one where you enter coding to produce a website and to edit coding and the second panel shows you what the website will look like and you can also edit here. This is way better then notepad and notepad++ but this program is dear and is a subscription based program.

Website Management System

SquareSpace

SquareSpace is a website management system which you can go into and just design the layout of a website and then this produces all of the coding itself for the website. For this system you need to pay a subscription ever month and they will uphold the website to your designs and needs.

Canva

Canva

Canva is a program which you can use to design your website style and design it off templates created for you and then this provides the coding for your website. To use this program, some of it is free but most of it is subscription based per annual payment
